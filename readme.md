## Tooltip componente vuejs totalmente customizavél

### Como usar?

- Na prop text você pode passar o texto do tooltip.
- Coloque dentro do  <tooltip> </tooltip> contéudo que irá ativar o tooltip.
 ``` html
  <tooltip text="Github">
      <img :src="github.png" alt="github"/>
   </tooltip>
  ```
